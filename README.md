# Redeemed - Faith-Based Accountability App

## Quick Start

1. Download all three files: `index.html`, `style.css`, and `script.js`
1. Keep them in the same folder
1. Open `index.html` in your browser
1. For GitHub Pages: Upload all three files to your repository

## File Structure

```
redeemed/
├── index.html   (Main HTML structure)
├── style.css    (All styles)
└── script.js    (All JavaScript)
```

## Features

- ✅ Streak tracking
- ✅ Daily checklist
- ✅ Panic button with breathing exercises
- ✅ Scripture memory
- ✅ Progress tracking
- ✅ Dark/light mode
- ✅ 100% client-side (no server needed)
